2、开发环境和剖析第一个C语言

2.1 程序创建4步骤

  1. 编辑

    编辑过程就是创建和修改 C 程序的源代码，linux常用编辑器为vim

  2. 编译

    编译器将源码转换机器语言，编译过查找错误并报告。编译输出结构为对象代码，存放文件为对象文件。linux扩展名为.0，windows为.obj“ gcc -o 1-1 1-1.c ”命令包含编译也包含链接，生成可执行文件1-1

  3. 链接

    链接器将源代码文件中由编译器产生的各种对象模块组合起来，再从 C 语言提供的程序库中添加必要的代码模块，将它们组合成一个可执行文件。链接器也可以检测和报告错误，例如程序中引用了一个根本不存在的库组件。

  4. 执行

    执行阶段就是成功完成了前述的三个过程后，运行程序。但是这个阶段可能会出现各种错误，包括输出错误，计算机什么也不做哦，甚至是计算机崩溃。

2.2语言的简单结构

  1. 预处理指令

    #include <stdio.h> C语言中的标准输入输出库，’#‘程序执行前预处理操作,编译器编译开始前预先处理指令(相当于加载模块 stdio.h)

  2. main(函数)

    main() 函数是“主函数”。每个 C 程序都由一个或多个函数组成，但每个 C 程序都必须有一个 main() 函数——因为每个程序总是从这个函数开始执行。

  3. 程序框架

    #include <stdio.h>
int main()
{
    XXXX 
    return 0;
}

  4. printf()

    printf() 是 C 编译系统提供的函数库中的输出函数。
